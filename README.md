# Virtual Assistant (Python)

This is a Python-based Virtual Assistant project that can perform basic tasks like:
- Opening applications
- Searching on Google
- Playing music
- Telling time and date
- Responding to voice commands

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser
- OS module

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the file:
   python main.py
